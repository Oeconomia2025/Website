Lead Developer Vesting Details

The lead developer allocation features a conservative vesting schedule with 3% of total supply locked initially. After a 6-month initial lock period, tokens unlock at 0.5% of total supply every 3 months, ensuring sustained commitment over the critical early development phases.

Governance Model

The OECgrd (OEC Guardian) system transforms standard token holding into active network participation with enhanced voting power and reward mechanisms.

🛡️ OECgrd Guardian System

Token Conversion
1:1
OEC tokens are wrapped 1:1 to OECgrd tokens, which can be unwrapped at any time (subject to any active lock periods).
Voting Power
Enhanced
OECgrd holders gain voting power multipliers based on lock duration: Standard (1x), Enhanced (1.5x), Elite (2x).
Network Security
Active
Staked OECgrd contributes to cross-protocol security through economic commitment and validator participation.
Fee Rewards
Proportional
Active stakers receive proportional rewards from transaction fees and protocol revenue.

Token Utility

OEC tokens serve multiple functions across the Oeconomia ecosystem, creating sustained demand and value accrual mechanisms.

🗳️ Governance Rights
Vote on protocol upgrades, parameter changes, treasury allocation, and cross-protocol coordination through OECgrd staking.
💰 Fee Revenue Sharing
Stakers receive proportional shares of transaction fees, protocol revenue, and yield generated across all Oeconomia protocols.
🛡️ Network Security
OECgrd staking provides economic security for the entire protocol suite, with rewards for maintaining network integrity.
🎯 Protocol Access
Enhanced features, reduced fees, and priority access to new protocol launches for OECgrd holders.
💎 Liquidity Incentives
Rewards for providing liquidity across Eloqura AMM pools and participating in cross-chain bridge operations.
🚀 Ecosystem Growth
Referral rewards, developer grants, and community building incentives to accelerate ecosystem adoption.

Value Accrual Mechanisms

The tokenomics design creates multiple value accrual streams:

  • Fee Burn: Portion of transaction fees permanently removed from circulation
  • Staking Demand: Enhanced rewards drive token locking and reduce circulating supply
  • Cross-Protocol Synergy: Integrated utility across Alluria, Eloqura, Artivya, and Iridescia
  • Governance Premium: Voting power concentration increases token value for serious participants
  • Revenue Distribution: Direct profit sharing with long-term stakers
